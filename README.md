# Homebrew Tap for Chris Edwards' Tools

This is a [Homebrew](https://brew.sh/) tap for installing CLI tools developed by Chris Edwards.

## Installation

### Install a formula

```bash
# Install abacus
brew install ChrisEdwards/tap/abacus
```

### Tap this repository

```bash
# Tap the repository first
brew tap ChrisEdwards/tap

# Then install formulas without the tap prefix
brew install abacus
```

## Available Formulas

### abacus

A TUI (Terminal User Interface) viewer for [Beads](https://github.com/beadstudio/beads) issue tracking.

**Installation:**
```bash
brew install ChrisEdwards/tap/abacus
```

**Repository:** https://github.com/ChrisEdwards/abacus

**Usage:**
```bash
abacus --help
abacus --version
```

## Updating Formulas

Formulas in this tap are automatically updated when new releases are published via GitHub Actions.
